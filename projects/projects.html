Programming Side Projects

  • Dose Response Analysis Tool
  • This project simplified the data graphing process for dose response curves in a pre-clinical testing lab. Before I wrote this the lab had to manually calculate and graph data from an excel file that contained over 3000 lines per experiment!
  • Surgical Scheduling Tool
  • This project was also a simple workplace automation tool. I used regular expressions to match the name of surgical types so that we could filter out surgeries that would not qualify for our study. Before this, somebody had to manually dig through an excel file.
  • Baseball Prediction and Simulation Engine
  • This project uses real world data to create a probabilistic model that can predict the performance of MLB players. This project is still in development, but is close to being ready to utilize for the upcoming 2020 season. Given data, the model can predict how well players will do based upon their previous stats and data.
  • College Football Simulation Game
  • This project was started November 2019, with a plan to release on steam at some point in late 2020. The game puts you in the position of coach. You decide who to recruit and how to gameplan to take your team to the very top of the league! This is a passion project that Fredrick has wanted to do since the early days of his computer programming obsession. Fredrick loves sports management sims and has always wanted to play a good college football one!
